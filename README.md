# Dotfile Installer

Concept configruation for bash on a new mac setup.

Inspired by many, originally forked from: holman/dotfiles
@holman does dotfiles

## Overview
The intention is to create a simplified version of dotfiles for bash. I wanted to remove anything that does not relate closely to bash as it makes it easier to transfer. 

This assumes certain packages have been installed (by homebrew). Likely, I will manually download the zip: https://github.com/smacpherson64/macos-setup and use this as the prerequsities.
